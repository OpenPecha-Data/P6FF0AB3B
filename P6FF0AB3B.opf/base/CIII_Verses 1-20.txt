sems can kun gyi ṅan soṅ gi | | sdug bsṅal ṅal bso’i dge ba daṅ | |byaṅ cub rgyur ’gyur dge
bsags pa | | de la rjes su yi raṅ ṅo | |(66)

lus can ’khor ba’i sdug bsṅal las | | ṅes par thar la yi raṅ ṅo | |skyob pa rnams kyi byaṅ cub
daṅ | | rgyal sras la yaṅ yi raṅ ṅo | |(67)

sems thams chad bde mdzad pa’i | | thugs bskyed dge ba rgya mtsho daṅ | |sems can phan par mdzad
pa la | | dga’ bas rje su yi raṅ ṅo | |(68)

phyogs rnams kun gyi saṅs rgyas la | | thal mo sbyar te gsol ba ni | |sems can sdug bsṅal mun
’thom la | | chos kyi sgron ma spar du gsol | |(69)

rgyal ba mya ṅan ’da’ bźed la | | thal mo sbyard ciṅ gsol ba ni | |’gro ’di mdoṅs par myi dgod
ciṅ | | bskal pa graṅs myed bźugs par gsol | |(70)

de ltar ’di dag kun byas ste | | dge ba bdag gis bsags pa gaṅ | |des ni sems can thams chad
kyi | | sdug bsṅal thams chad bsal bar śog | |(71)

’gro ba nad pa ci srid du | | nad sos gyur gyi bar du ni | |sman daṅ sman pa ñid dag daṅ | |
de’i nad g-yag byed par śog | |(72)

zas daṅ skom gyi char phab ste | | bkres daṅ skom pa’i gnod pa bsal | |mu ge’i bskald pa bar
ma’i tshe | | bdag ni zas daṅ skom du gyurd | |(73)

sems can phoṅs śiṅ dbul ba la | | bdag ni myi zad gter gyurd te | |yo byad mkho dgu sna tshogs
su | | mdun du ñe bar gnas gyur cig | |(74)

lus daṅ de bźin loṅs spyod daṅ | | dus sum dge ba thams chad kyaṅ | |sems can kun gyi don ’grub
phyir | | phaṅs pa myed par gtaṅ bar bya | |(75)

thams chad btaṅ bas mya ṅan ’da’ | | bdag blo mya ṅan ’das pa sgrub | |thams chad gtaṅ du chabs
gcig ma1 | | sems chan rnams la gtaṅ ba mchog | |(76)

bdag gis lus can thams chad la | | lus ’di ci bder byin zin kyis | |rtag du gsod daṅ smod pa
’am | | brdeg stsogs ci dga’r byed la rag | |(77)

bdag gi lus kyis rtse byed dam | | co ’dri gźa’i[(sic??)] rgyu byed kyaṅ | |bdag gi lus ’di
byin zin gyis | | ’di’i kha das ci źig bya | |(78)

de la gnod par myi ’gyur ba’i | | las gaṅ yin pa ’aṅ byed du chug | |bdag la dmyigs nas nam
du yaṅ | | ’ga’ yaṅ don myed ma gyur cig | |(79)

bdag la dmyigs nas gaṅ dag gi | | khro ba ’am dad pa’i sems byuṅ ba | |de ñid rtag du de dag
gi | | don kun ’grub pa’i rgyur gyur cig | |(80)

gaṅ dag bdag la kha zer ram | | gźan gaṅ gnod par byed pa ’am | |de bźin phyar ka gtoṅ ba yaṅ
| | thams chad byaṅ cub skal ldan gyurd | |(81)

bdag ni mgon myed rnams kyi mgon | | lam źugs rnams kyi ded pon daṅ | |rgal ’dod rnams kyi gru
daṅ ni | | gziṅs daṅ zam pa ñid du ’gyur | |(82)

gliṅ don gñer la gliṅ daṅ ni | | gnas mal ’dod la gnas mal daṅ | |bdag nil us can bran ’dod
pa | | kun gyi bran du gyur par śog | |(83)

yid bźin nor daṅ bum pa bzaṅ | | rig sṅags grub daṅ sman chen daṅ | |dpag bsams kyi ni śiṅ dag
daṅ | | lus can rnams kyi ’dod bźor gyurd | |(84)

sa stsogs ’byuṅ ba chen po daṅ | | nam mkha’ bźin du rtag par yaṅ | |sems can dpag tu myed pa
yi | | rnam maṅ ñer ’tsho’i gźir yaṅ śog | |(85)
